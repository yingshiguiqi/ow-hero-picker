# 守望先锋英雄选择助手

> 帮你在排位和竞技中，快速选出「这把最合适」的英雄。

- **最后更新**：2025-12-12 14:06 (UTC+8)
- **平台**：Windows 桌面版 (WPF) ＋ Web 网页版 (Next.js)
- **适合人群**：常打排位、需要快速选将/复盘阵容的守望先锋玩家

---

## 🎯 项目简介

本项目是一个针对《守望先锋》的**英雄选择与阵容分析工具**，提供：

- 不同地图下的**推荐英雄预设**
- 实时的**敌我阵容分析与补位建议**
- 全英雄的**克制关系查询**
- 支持**自定义预设导出/导入**，方便分享给队友

你可以把它当作：

- 自己的「**地图英雄小抄**」
- 一套「**阵容评估 + 克制索引**」

---

## 🖥️ WPF 桌面版（推荐）

基于 .NET 8 + WPF 的原生 Windows 应用，启动和交互都更流畅，是推荐使用的版本。

### 运行环境

- Windows 10 / 11
- .NET 8 SDK 或更高版本

### 快速开始

```bash
cd OwHeroPicker
 dotnet run
```

首次运行会自动加载内置的地图与英雄预设配置。

### 主要功能

#### 📍 地图英雄推荐

- 覆盖**全部现役地图**：
  - 抢点图、机动推进图、闪点图、纯推车图、混合推车图
- 每张地图 / 子地图都有独立预设
- 三级评级：
  - **不错哦**：强烈推荐
  - **一般般**：可用但看阵容
  - **不好用**：当前环境不太建议
- 预设管理：
  - 右键英雄即可编辑评级
  - 一键导出 / 导入 JSON 配置
  - 可重置为官方预设

#### ⚔️ 英雄克制关系

- 为每个英雄维护一组**被谁克制 / 克制谁**的信息
- 提供简短文字说明（为什么被克、注意点）
- 点击英雄即可跳转查看对应克制列表
- 支持外号/拼音搜索，例如：
  - `科比` → 托比昂
  - `大锤` → 莱因哈特
  - `法鸡` → 法老之鹰
  - `猩猩` → 温斯顿

#### 🎮 实战阵容分析

- 5 个友方槽位 + 5 个敌方槽位
- 根据敌方阵容实时推荐可选英雄
- 简单的阵容体检：
  - 是否缺主坦 / 副坦
  - 输出位是否过于单一
  - 治疗量和保人能力是否不足

#### 🦸 英雄数据面板

- 收录所有现役英雄：坦克、输出、支援
- 每个英雄包含：职业、难度、别名、头像路径等
- 所有头像本地存储，避免在线加载带来的卡顿

### 技术实现

| 类别   | 技术                         |
| ------ | ---------------------------- |
| 框架   | .NET 8 + WPF                 |
| 架构   | MVVM (CommunityToolkit.Mvvm) |
| UI     | 自定义深色主题 & 渐变标题栏 |
| 数据   | C# 静态数据 + JSON 配置      |

## 🌐 Web 网页版

Web 版主要用于**快速分享和在线体验**，功能与桌面版类似，但体验略逊一筹。

> 源码位于 `web/` 目录，在线 Demo 使用 GitHub Pages 部署。

### 在线体验地址

- https://yingshiguiqi.github.io/ow-hero-picker/

### 本地运行

```bash
cd web
npm install
npm run dev
# 访问 http://localhost:3000
```

### 前端技术栈

| 类别     | 技术                    |
| -------- | ----------------------- |
| 框架     | Next.js 14 + React 18   |
| 状态管理 | Zustand                 |
| UI       | TailwindCSS + shadcn/ui |
| 工具     | TypeScript, ESLint 等   |

## 📁 仓库结构一览

```text
ow/
├── OwHeroPicker/          # WPF 桌面版主工程
│   ├── Assets/Heroes/     # 英雄头像资源
│   ├── Converters/        # WPF 数据转换器
│   ├── Data/              # 英雄 / 地图 / 克制静态数据
│   ├── Models/            # 数据模型
│   ├── Services/          # 业务服务（预设、推荐等）
│   ├── Themes/            # 主题样式（暗色主题等）
│   ├── ViewModels/        # MVVM 视图模型
│   └── Views/             # XAML 视图
├── web/                   # Web 版源码
├── 预设配置-preset.json    # 默认地图英雄预设
└── README.md
```
